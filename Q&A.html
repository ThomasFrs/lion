Where do I download the game?

You can download the game now by hitting "Back" on the left pannel, then Click on "ThroneRoom.exe".

How do I run it?

Run the game by clicking on throneroom.exe in the folder where you downloaded it.

Is there any risk of malware infection of damage on my computer?

No, Windows defender restrains small developpers from publishing games with "Unknown Publisher" feature.

Only Windows OS is capable of running this file.
